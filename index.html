Learn HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. Below are some basic HTML elements:

1. Basic Structure

Every HTML document starts with a <!DOCTYPE html> declaration, followed by the <html> element:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

The <!DOCTYPE html> declaration defines the document type and version of HTML. The <html> element is the root element of an HTML page. The <head> element contains meta-information about the document, and the <body> element contains the content of the document.

2. Headings

HTML headings are defined with the <h1> to <h6> tags:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

<h1> defines the most important heading, and <h6> defines the least important heading.

3. Paragraphs

Paragraphs are defined with the <p> tag:

<p>This is a paragraph.</p>

4. Links

Links are defined with the <a> tag:

<a href="https://www.example.com">This is a link</a>

The href attribute specifies the URL of the page the link goes to.

5. Images

Images are defined with the <img> tag:

<img src="image.jpg" alt="Description of image">

The src attribute specifies the path to the image, and the alt attribute provides an alternative text for the image.

6. Lists

HTML supports ordered and unordered lists:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

<ul> defines an unordered list, and <ol> defines an ordered list. List items are defined with the <li> tag.

7. Tables

Tables are defined with the <table> tag:

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

<tr> defines a table row, <th> defines a table header, and <td> defines a table cell.

8. Forms

Forms are used to collect user input:

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    <br>
    <input type="submit" value="Submit">
</form>

The <form> element defines a form, and the action attribute specifies where to send the form data when submitted. The method attribute specifies the HTTP method to use when sending form data.

9. Semantic Elements

HTML5 introduced semantic elements to define different parts of a web page:

<header>Header content</header>
<nav>Navigation links</nav>
<main>Main content</main>
<footer>Footer content</footer>

These elements provide meaning to the structure of the web page, making it easier to read and maintain.

10. Audio and Video

HTML5 also introduced elements for embedding audio and video:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

<video controls>
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video element.
</video>

The <audio> and <video> elements allow you to embed media files with built-in controls for play, pause, and volume.

11. Inline Frames

Inline frames (<iframe>) are used to embed another document within the current HTML document:

<iframe src="https://www.example.com" width="600" height="400">
    </iframe>

The src attribute specifies the URL of the document to embed.

12. Meta Tags

Meta tags provide metadata about the HTML document:

<meta charset="UTF-8">
<meta name="description" content="Learn HTML">
<meta name="keywords" content="HTML, CSS, JavaScript">
<meta name="author" content="Your Name">

The charset attribute specifies the character encoding for the HTML document. The name and content attributes provide additional metadata.

13. Scripts

Scripts are used to add interactivity to your web pages:

<script>
    function showAlert() {
        alert("Hello, world!");
    }
</script>
<button onclick="showAlert()">Click me</button>

The <script> element is used to define client-side JavaScript. The onclick attribute is used to execute JavaScript when a button is clicked.

14. Canvas

The <canvas> element is used to draw graphics on the fly via JavaScript:

<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
    var canvas = document.getElementById("myCanvas");
    var ctx = canvas.getContext("2d");
    ctx.fillStyle = "#FF0000";
    ctx.fillRect(0, 0, 200, 100);
</script>

The getContext("2d") method returns a drawing context on the canvas.

15. SVG

SVG (Scalable Vector Graphics) is used to define vector-based graphics:

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

SVG elements and attributes are used to draw shapes like circles, rectangles, and paths.
